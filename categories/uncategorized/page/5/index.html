Kereta Api ke Bogor - Jadwal, Rute, Harga Tiket, & Info KA

Fri Jun 10, 2022 by Reinaldo Massengill in  Uncategorized

Stasiun Bogor merupakan stasiun kereta api kelas besar tipe A yang terletak di Cibogor, Bogor Tengah, Bogor. Pada masa kolonial Belanda bernama Station Buitenzorg. Stasiun Bogor dibangun oleh Nederlandsch-Indische Spoorweg Maatschappij pada tahun 1872 sebagai titik akhir jalur kereta api Batavia-Weltevreden-Depok-Buitenzorg. Stasiun ini dibuka untuk pertama kalinya untuk umum pada 31 Januari 1873. Stasiun ini menjadi andalan para wisatawan yang ingin menghabiskan liburan akhir pekan di kota Bogor. Stasiun yang dibangun pada tahun 1881 ini menjadi stasiun terminus kereta api untuk perjalanan KRL Commuter Line yang melayani kawasan Jabodetabek, yakni menuju Stasiun Jakarta Kota, Stasiun Angke, Stasiun Kampung Bandan, dan Stasiun Jatinegara.
