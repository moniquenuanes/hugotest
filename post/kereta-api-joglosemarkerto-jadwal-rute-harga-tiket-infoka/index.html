Kereta Api Joglosemarkerto - Jadwal, Rute, Harga Tiket, &amp; InfoKA

May 2022 · 3 minute read

Kereta api Joglosemarkerto merupakan kereta api kelas eksekutif dan ekonomi AC plus yang dioperasikan PT Kereta Api Indonesia di Jawa yang melayani rute lingkar Solo - Yogyakarta - Purwokerto - Semarang dan sebaliknya.

Total jarak tempuh KA ini mengelilingi jalur rel di wilayah Jawa Tengah adalah 579 km dengan total waktu tempuh 11 jam 25 menit. Bila dilihat dari papan peta pemberhentian kereta api di dalam rangkaian, kereta ini biasa juga disebut Kereta api Kamandaka khusus untuk relasi Semarang - Purwokerto.

Kereta api yang diresmikan pada 1 Desember 2018 ini, bertujuan melebur dua layanan kereta api yang telah ada sebelumnya, yaitu Kamandaka dan Joglokerto. Hal ini dilakukan demi penyederhanaan pelayanan sekaligus meningkatkan okupansi rute Yogyakarta – Semarang tanpa perlu transit atau berganti kereta api lain di Stasiun Purwokerto dan Solo Balapan.

Meski dilebur, kereta ini sama sekali tidak ada hubungannya dengan Kamandaka, Joglokerto, maupun Joglosemar; ketiganya merupakan layanan tersendiri.

Untuk tarif atau harga tiket KA Joglosemarkerto terbagi dalam berbagai macam golongan. Besaran tarifnya disesuaikan dengan jarak tempuh dan jenis layanan kereta, kelas eksekutif atau kelas ekonomi.

Asal Nama

Nama kereta api Joglosemarkerto merupakan kependekan dari Jogja-Solo-Semarang-Purwokerto. Nama Joglosemarkerto juga ingin menjelaskan bahwa kereta ini menghubungkan kota-kota di Jawa Tengah sisi selatan dan utara, sekaligus Yogyakarta.

Sejarah Kereta

Meskipun telah ada kereta api yang melayani rute Semarang–Solo (Kalijaga), Solo–Purwokerto (Joglokerto), dan Purwokerto–Semarang (Kamandaka), namun intensitas perjalanan kereta api tersebut masih tergolong sedikit.

Sebagian besar penumpang juga harus transit terlebih dahulu di stasiun-stasiun besar di lingkar Jawa Tengah. Disebabkan tingginya permintaan untuk rute ini, PT KAI kemudian membuat sebuah layanan baru untuk mengakomodasi pelayanan kereta api di lingkar Yogyakarta–Purwokerto–Tegal–Semarang–Solo dalam format melingkar. Nama Joglosemarkerto mulai digaungkan pada pertengahan November 2018 hingga kini.

Rangkaian dan Fasilitas Kereta

Kereta api Joglosemarkerto memiliki 3 rangkaian kereta api dengan masing-masing terdiri dari 2 kereta eksekutif dan 7 kereta ekonomi. Total penumpang yang dapat diangkut oleh kereta ini dalam setiap perjalanannya adalah sebanyak 660 penumpang. Fasilitas kereta ini meliputi AC, toilet, bagasi, colokan listrik, serta gantungan.
