Kereta api Bima dihadirkan oleh PT KAI sebagai layanan kereta api kelas eksekutif satwa untuk jurusan Jakarta – Malang. Kereta api ini melayani perjalanan dua arah dengan titik keberangkatan dari Stasiun Jakarta Gambir dan titik perhentian terakhir di Stasiun Malang (Kotabaru). Frekuensi perjalanan KA Bima tersedia sebanyak satu kali sehari dengan jarak tempuh 907 kilometer. Dengan kecepatan rata-rata 60 – 120 km per jam, KA Bima akan membawa penumpangnya mencapai Malang dari Jakarta, atau sebaliknya, dalam durasi perjalanan sekitar 15,5 jam. Read more →
