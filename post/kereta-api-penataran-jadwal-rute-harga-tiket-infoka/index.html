Kereta Api Penataran - Jadwal, Rute, Harga Tiket, & InfoKA

Tue May 17, 2022 in  Uncategorized

Kereta Api Penataran merupakan KA lokal yang melayani rute Surabaya Kota – Malang – Blitar. Kereta api ini mulai beroperasi sejak 14 januari 1971 di bawah naungan Daerah Operasi VIII Surabaya. Kereta ini melayani kelas jenis ekonomi AC.

Kereta yang memiliki lebar sepur 1067 mm ini mampu menempuh kecepatan 60 – 70 km/jam. Setiap rute, KA Penataran akan melalui 21 – 23 stasiun pemberhentian. Mulai dari Surabaya Kota (kecuali KA 436), Surabaya Gubeng, Wonokromo, Waru, Gedangan, Sidoarjo, Tanggulangin, Porong, Bangil, Lawang, Singosari, Blimbing, Malang, hingga stasiun lainnya. Di stasiun Pohgajih, KA 431 akan menuju arah barat, sedangkan KA 436 dan KA 434 ke arah timur. Dalam satu hari, Kereta Api Penataran melayani 4 kali keberangkatan asal Surabaya dan Blitar.

Tiket Kereta Api Penataran tersedia dalam beberapa pilihan, tergantung rute yang dipilih mulai dari Rp10.000 – 15.000. Dengan harga tersebut, penumpang sudah mendapatkan fasilitas gerbong AC dengan kursi berhadapan 2 - 2. Selain itu, KA juga dilengkapi gerbong penyedia makanan khusus para penumpang.

Asal nama Kereta Api Penataran diambil dari nama candi yang berada di Blitar. Konon, candi tersebut merupakan peninggalan Raja Kertajaya (Srengga) asal Kerajaan Kediri.

Setibanya di Blitar, KA penataran berubah nama menjadi KA Rapih Dhoho dan melanjutkan perjalanan menuju Surabaya melalui Kediri dan Kertosono.
